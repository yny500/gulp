# **Gulp Build Setup**
  
#### 💡 Vanilla JS 프로젝트 개발을 위한 Gulp 기반 빌드 환경입니다. 
#### 반복적인 작업을 자동화하기 위해 작업하였습니다.
<br>

## 주요 기능
- SCSS → CSS 컴파일 및 minify
- JavaScript 번들링 및 압축
- 이미지 최적화
- HTML 템플릿 처리
- 자동 리로드
<br>

## 기술 스택
- Gulp
- Babel
- SCSS
<br>

## 구현 포인트
- 반복적인 프론트엔드 작업 자동화
- 개발 생산성 향상을 위한 빌드 파이프라인 구성
- 프로젝트에 바로 적용 가능한 구조 설계
